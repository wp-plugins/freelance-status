== Description ==

Sidebar-widget displaying your freelance-availability status in a nice box. Might be used for other stuff as well. You can set options (color and text) for two states (A and B) and switch comfortably between them. The widget is rendered as a nice box with a slight arrow in the sidebar.

== Installation ==

1. Upload the `freelance_status` folder to the `/wp-content/plugins/` directory
2. Add the newly available "Freelance Status" widget to any sidebar.
3. Set it's options.
