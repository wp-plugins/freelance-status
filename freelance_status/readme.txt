== Description ==

Sidebar-widget displaying your freelance-availability status (nicely). Might be used for other stuff as well.

== Installation ==

1. Upload the `freelance_status` folder to the `/wp-content/plugins/` directory
2. Add the newly available "Freelance Status" widget to any sidebar.
3. Set it's options.
